# Base Task: Fashion-MNIST Classifier

This folder contains the implementation of a fully connected Neural Network using PyTorch to classify the Fashion-MNIST dataset.

## Final Training Metrics
* **Final Validation Accuracy:** 85.57%
* **Final Validation Loss:** 0.4072

## Directory Contents
* `/notebooks/`: Contains the Jupyter/Colab notebook with the full training pipeline, forward/backward propagation, and matplotlib plots.
* `/saved_models/`: Contains the saved PyTorch model weights (`.pkl`).
* `submission.csv`: The generated predictions on the test dataset.
